# 과학 문제 생성기

과학 교재 PDF를 분석해 AI 기반으로 문제를 생성하는 웹 앱입니다.  
이 버전은 순수 HTML/CSS/JavaScript로 구성된 정적 사이트입니다.

## 페이지 구성

| 파일 | 설명 |
|------|------|
| `index.html` | PDF 업로드 |
| `settings.html` | 단원 선택 및 생성 설정 |
| `editor.html` | 문제 편집 |
| `storage.html` | 문제 저장소 |
| `css/style.css` | 공통 스타일 |
| `js/app.js` | 공통 JavaScript 로직 |

## 기능

- PDF 업로드 (드래그 앤 드롭 지원)
- 단원별 문제 생성 시뮬레이션
- 객관식 / 서술형 / 혼합 문제 유형
- 문제 편집 및 재생성
- 로컬 스토리지에 문제 저장/관리
- 단원별 필터링

## 사용법

`index.html`을 브라우저에서 열거나, GitHub Pages로 배포하세요.
